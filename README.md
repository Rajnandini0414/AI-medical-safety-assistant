# AI Medical Safety Assistant

An AI-based system that helps monitor and ensure medical safety by analyzing patient data and detecting potential risks.

## Features
- Detects abnormal health indicators
- Provides early safety alerts
- Helps healthcare professionals monitor patients

## Technologies Used
- Python
- Machine Learning
- Flask / FastAPI
- Pandas / NumPy
